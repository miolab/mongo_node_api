# MongoDB & Node.js

RestAPI (Node.js &amp; MongoDB)

- version

  ```
  $ npm --version
  6.14.4

  $ mongo --version
  MongoDB shell version v4.2.5
  ```

  - MongoDB installed by Homebrew

---

## setting

```
$ npm init -y
```

### install express

```
$ npm install express
```

### init confirmation Server running

- http://localhost:3000/

  ```
  $ node app.js
  ```

### install nodemon

```
$ npm install nodemon
```

Continuous execute Server running:

- http://localhost:3000/

  ```
  $ npx nodemon app.js
  ```

### install Mongoose

```
$ npm install mongoose

$ brew services start mongodb-community
```
